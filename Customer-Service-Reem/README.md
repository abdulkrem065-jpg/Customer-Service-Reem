# 🤖 وكيل ريم للخدمة الذاتية

وكيل ذكاء اصطناعي للرد على المكالمات والرسائل خارج أوقات العمل.

## النشر السريع على Render
1. ارفع المشروع إلى GitHub
2. سجل في render.com
3. اختر "New Web Service" واربط المستودع
4. أضف متغير البيئة: GEMINI_API_KEY
5. اضغط "Create"

## المتغيرات البيئية
- GEMINI_API_KEY: مفتاح API من Google AI Studio
